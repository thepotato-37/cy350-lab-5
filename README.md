# CY350 Lab 5 Quarto Website

This folder contains a simple Quarto website for In-Class Lab 5.

## Preview

```bash
quarto preview
```

## Render

```bash
quarto render
```

The static site is rendered to `docs/`.

## Publish With GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the source to the `main` branch and the `/docs` folder.
4. Save the settings and submit the generated Pages URL.

You can also run:

```bash
quarto publish gh-pages
```
